\begin{abstract}
This paper presents significant advancements in the Unified Applicable Time (UAT) framework, demonstrating its robust validation against observational cosmology data. Building upon our previous work on primordial black hole (PBH) evolution, we extend UAT to comprehensive cosmological applications using 30 Hubble parameter measurements $H(z)$ across redshift range $z = 0.07$ to $z = 1.75$. Our analysis reveals statistical equivalence with $\Lambda$CDM cosmology ($\chi^{2}_{\mathrm{UAT}} = 14.8$ vs $\chi^{2}_{\Lambda\mathrm{CDM}} = 14.8$, $\chi^{2}_{\mathrm{red}} = 0.528$) while establishing a theoretically motivated framework for PBH mass-dependent corrections. The UAT framework incorporates loop quantum gravity corrections through the Barbero-Immirzi parameter ($\gamma = 0.2375$) and provides a foundation for controlled modifications, though current observational data show minimal deviations from $\Lambda$CDM. This work establishes UAT as a conservative, observationally consistent extension of standard cosmology with applications to cosmological tensions and quantum gravity phenomenology.
\end{abstract}

\section{Introduction}

The Unified Applicable Time (UAT) framework was initially developed to address the multi-scale temporal challenges in primordial black hole (PBH) evolution simulations, integrating cosmological expansion, gravitational time dilation, and loop quantum gravity (LQG) corrections. Our previous work demonstrated UAT's effectiveness in modeling PBH evaporation while maintaining physical consistency across different regimes.

This enhanced study presents a comprehensive cosmological validation of the UAT framework against observational Hubble parameter data, establishing its viability as a conservative extension of $\Lambda$CDM cosmology. The key advancements include:

\begin{itemize}
\item Comprehensive validation against 30 $H(z)$ measurements from $z = 0.07$ to $z = 1.75$
\item Statistical demonstration of equivalence with $\Lambda$CDM cosmology
\item Theoretical framework for PBH mass-dependent corrections with minimal current impact
\item Professional implementation with robust error handling and visualization
\item Extended applications to cosmological parameter estimation
\end{itemize}

The UAT framework represents a significant advancement in bridging quantum gravitational effects with observational cosmology, providing a mathematically consistent approach to extending standard cosmological models.

\section{Theoretical Framework}

\subsection{Unified Applicable Time Formulation}

The core UAT equation integrates cosmological, relativistic, and quantum effects:

\begin{equation}
t_{\mathrm{UAT}} = t_{\mathrm{event}} \times \frac{1}{a(t)} \times \sqrt{\max\left(1-\frac{2GM(t)}{c^{2} r},\left(\frac{l_{\mathrm{Planck}}}{r}\right)^{2}\right) \times \frac{1}{1 + \frac{\gamma l_{\mathrm{Planck}}^{2}}{4\pi r_{s}^{2}}}} + \frac{d_{L}}{c}
\end{equation}

where:
\begin{itemize}
\item $a(t)$: cosmological scale factor
\item $M(t)$: PBH mass evolution
\item $r_{s} = 2GM/c^{2}$: Schwarzschild radius
\item $l_{\mathrm{Planck}} = \sqrt{\hbar G/c^{3}}$: Planck length
\item $\gamma = 0.2375$: Barbero-Immirzi parameter (LQG)
\item $d_{L}$: luminosity distance
\end{itemize}

\subsection{Hubble Parameter Formulation}

For cosmological applications, we derive the UAT-corrected Hubble parameter:

\begin{equation}
H_{\mathrm{UAT}}(z, M) = H_{0} \cdot E_{\Lambda\mathrm{CDM}}(z) \cdot \left[1 + f(z, M)\right]
\end{equation}

with the correction factor:

\begin{equation}
f(z, M) = \gamma \cdot \ln\left(\frac{M}{M_{0}}\right) \cdot \ln(1+z)
\end{equation}

where $M_{0} = 10^{12}$ kg serves as the reference mass. The theoretical framework allows for corrections up to $\pm 113.7\%$, though physical bounds and current observational constraints limit practical deviations to $\pm 0.2\%$ in our conservative numerical implementation.

\subsection{Statistical Validation Framework}

We employ rigorous statistical comparison using the $\chi^{2}$ statistic:

\begin{equation}
\chi^{2} = \sum_{i=1}^{N} \left(\frac{H_{\mathrm{obs}}(z_{i}) - H_{\mathrm{model}}(z_{i})}{\sigma_{i}}\right)^{2}
\end{equation}

with reduced $\chi^{2}$ calculated as $\chi^{2}_{\mathrm{red}} = \chi^{2}/(N - k)$ where $N=30$ data points and $k=2$ parameters.

\section{Discussion}

\subsection{Physical Interpretation}

The UAT framework introduces several key physical insights:

\begin{enumerate}
\item \textbf{Conservative Foundation}: UAT maintains exact agreement with $\Lambda$CDM for current observational data, ensuring robust baseline compatibility while providing extension capabilities.

\item \textbf{Theoretical Extension Capability}: The framework provides a mathematically consistent mechanism for PBH mass-dependent corrections, with theoretical variations up to $\pm 113.7\%$ for extreme mass values.

\item \textbf{Numerical Stability}: Conservative bounds ($\pm 0.2\%$) in the numerical implementation prevent unphysical deviations while preserving the theoretical framework for future high-precision applications.

\item \textbf{Redshift Evolution}: The $\ln(1+z)$ factor provides natural scaling with cosmic time, with corrections becoming potentially significant at higher redshifts beyond current observational range.

\item \textbf{Theoretical Consistency}: The Barbero-Immirzi parameter establishes connection to loop quantum gravity foundations, providing physical motivation for the correction framework.
\end{enumerate}

\subsection{Numerical Implementation Strategy}

Our implementation prioritizes numerical stability and physical consistency:

\begin{itemize}

\item Smooth transition functions ensure numerical stability across parameter space
\item The framework is designed for future high-precision data where theoretical corrections may become observable
\item Current equivalence with $\Lambda$CDM validates the conservative approach for present observational constraints
\end{itemize}
